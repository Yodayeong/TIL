## 코드 블록

### Fenced Code Block

- `(backtick) 기호 3개를 활용하여 작성한다.
- 특정 언어를 명시하면 Syntax highlighting 기능이 적용된다.

```python
print('hello')

if True:
    print('t')
else
	print('f')
# 주석
```

```html
print('hello')
# 주석?
<h1>
    제목
</h1>
<!-- 주석 -->
```

### Inline Code Block

`(backtick) 기호 1개를 활용하여 작성한다.

`print` 는 파이썬에서 출력하는 함수이다.
