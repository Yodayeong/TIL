## 회전(이차원배열)

- 왼쪽으로 90도 회전하기

  ```bash
  matrix = [
  	[1, 2, 3],
  	[4, 5, 6],
  	[7, 8, 9]
  ]
  
  n = 3
  rotated_matrix = [[0] * n for _ in range(n)]
  
  for i in range(n):
  	for j in range(n):
  		rotated_matrix[i][j] = matrix[j][n-i-1]
  ```

  

- 오른쪽으로 90도 회전하기

  ```bash
  matrix = [
  	[1, 2, 3],
  	[4, 5, 6],
  	[7, 8, 9]
  ]
  
  n = 3
  rotated_matrix = [[0] * n for _ in range(n)]
  
  for i in range(n):
  	for j in range(n):
  		rotated_matrix[i][j] = matrix[n-j-1][i]
  ```

### 델타 탐색(Delta Search)

: (0, 0)에서부터 이차원 리스트의 모든 원소를 순회하며(완전탐색) 각 지점에서 	상하좌우에 위치한 다른 지점을 조회하거나 이동하는 방식

- 델타값을 이용해 상하좌우로 이동하기

  ```bash
  #1) 행을 x, 열을 y로 표현
  dx = [-1, 1, 0, 0]
  dy = [0, 0, -1, 1]
  
  #2)행을 r, 열을 c로 표현
  dr = [-1, 1, 0, 0]
  dc = [0, 0, -1, 1]
  
  #1. 델타값을 이용해 상하좌우 이동
  for i in range(4):
  	nx = n + dx[i]
  	ny = y + dy[i]
  	#2. 범위를 벗어나지 않으면 갱신
  	if 0 <= nx < 3 and 0 <= ny < 3:
  		x = nx
  		y = ny
  ```

- 이차원 리스트의 상하좌우 탐색 정리

  ```bash
  #1. 델타값 정의(상하좌우)
  dx = [-1, 1, 0, 0]
  dy = [0, 0, -1, 1]
  
  #2. 이차원 리스트 순회
  for x in range(n):
  	for y in range(m):
  		#3. 델타값을 이용해 상하좌우 이동
  		for i in range(4):
  			nx = x + dx[i]
  			ny = y + dy[i]
  			#4. 범위를 벗어나지 않으면 갱신
  			if 0 <= nx < n and 0 <= ny < m:
  				x = nx
  				y = ny
  ```
